Covid19 War - Gameplay Overview


Covid19 War is inspired by the classic game Space Invaders. Instead of defending Earth from alien invasions, players must fight against a viral infection spreading across the globe. Players need to shoot the viruses to prevent further damage using cures. Additionally, the game blends retro aesthetics with modern gameplay to create an engaging and educational experience.

Gameplay

  • Use the Left/Right/Up/Down arrow keys to move your character.
  • Press the Space bar to shoot the viruses.
  • Press Esc to pause the game.

Rules

  • Survive until the end of the game.
  • Defeat as many viruses as possible to achieve a high score.
  • The game ends if your health points (HP) drop to 0.

Artworks

The game's visual style is inspired by retro 8-bit graphics. All artwork was created using the Aseprite program to capture the nostalgic feel of classic arcade games.

Additional Notes

This project allowed me to explore game development principles such as function usage and program flow. I also learned how to organize and structure a game system effectively. By using my own artwork, I enhanced the visual appeal of the game. The experience gained from this project has been invaluable and has contributed significantly to my main project.
